### Atividade

## Traga 1 exemplo real (app mobile, web ou desktop) de aplicação de IA em um software.

- Software + problema (1 frase)
- Funcionalidade de IA (o que faz) + entrada/saída
- Técnica provável (ex.: classificação, recomendação, LLM/RAG, visão)
- Métrica de sucesso (como medir)

### Resposta

- Software + Problema: O GitHub Copilot é uma extensão para IDEs que resolve
  o problema na perda de produtividade com digitação de código repetitivo e
  a busca por sintáxes e documentações básicas.
- Funcionalidade de IA (o que faz) + entrada/saida: Ele atua como um programador
  virtual, segerindo trechos de código ou funções inteiras enquanto você digita.
  Entrada: O contexto do arquivo atual, variáveis ja declaradas e comentários.
  Saída: A sugestão em texto de um bloco de código funcional para autocompletar a instrução.
- Técnica provável: LLM otimizado e treinado especificamente em bilhões de linhas de
  código de repositórios públicos.
- Métrica de sucesso (como medir): Taxa de aceitação das sugestões e a redução no
  lead time.
